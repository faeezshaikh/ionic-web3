# ionic-web3

1. ``` ionic start ionic-web3 sidemenu ```

2. Added in package.json
   ``` "web3": "^0.17.0-alpha" ```

3. ``` npm i  ```

4. ``` ionic generate providers web3service ```

5. Write the code in Service.
6. Inject service in controllers.
